<div align="center">
  <h1>ChartJS++</h1>

  <p>A wrapper of Chart.JS enabling you to configure your graphs with OO interfaces.</p>
</div>

## Build

- `npm run build`
- `npm run build-prod`
- `npm run watch`

## Generate lib .d.ts

`dts-bundle-generator -o ./dist/dev/libs/ChartJS/index.d.ts src/libs/ChartJS/index.ts`
